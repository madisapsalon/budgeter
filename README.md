## Budgeter API

Example API written with [Nest](https://github.com/nestjs/nest) framework and uses MySQL database.

Example API runs here https://budgeter-api.herokuapp.com/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
